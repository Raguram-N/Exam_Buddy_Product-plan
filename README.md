# 🎓 ExamBuddy AI - AI Classroom Intelligence Platform

ExamBuddy AI helps students learn through voice conversations and helps teachers understand what students actually know.

---

## 📱 Student Features

| Feature | Description |
|----------|------------|
| ⚡ Revise | Quick chapter summaries, key concepts, and exam notes. |
| 📚 Ask Doubt | Students ask questions using voice or text and receive instant explanations. |
| 🧠 Practice | Topic-based questions with feedback and explanations. |
| 🎙️ Voice Mentor | Natural voice conversations with an AI learning mentor. |
| 📊 Session Summary | Learning Score, Strong Areas, Weak Areas, and Recommended Topics after every session. |

---

## 👨‍🏫 Teacher Features

| Feature | Description |
|----------|------------|
| 🎯 Class Readiness | See which students are ready for upcoming exams. |
| 🔍 Learning Gap Detection | Identify concepts students struggle with most. |
| 🚨 Student Attention Center | Detect low-engagement or at-risk students. |
| 📝 Revision Assignment | Assign revision topics in seconds. |
| ⚡ Recommended Actions | AI-generated suggestions such as revision reminders and targeted practice recommendations. |
